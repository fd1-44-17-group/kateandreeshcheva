$ bem make

Первая сборка bem-tools займет некоторое время, так как именно в этот момент в директорию проекта устанавливается все необходимые npm-пакеты.

Для запуска сервера выполняем:

$ bem server

В результате вы увидите сообщение:

info: Server is lidtening on port 8080. Point your browser to http://localhost:8080

Проблема?

Ели порт 8080 уже используется другой программой, его можно переназначить с помощью аргумента -p.
